# ⚡ Electric Vehicle Data Analysis Dashboard

An interactive Tableau dashboard designed to analyze Electric Vehicle (EV) adoption trends, vehicle distribution, manufacturer performance, and market insights through data visualization.

## 📊 Features

* EV Adoption Trend Analysis
* Manufacturer-wise Vehicle Distribution
* Electric Range Analysis
* State-wise EV Registration Insights
* Vehicle Type Comparison
* CAFV Eligibility Analysis
* Interactive Filters and Dashboards
* Data-Driven Visualizations

## 📈 Key Insights

* Analyzed growth trends in Electric Vehicle adoption.
* Identified top-performing EV manufacturers.
* Examined vehicle distribution across states and regions.
* Compared Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs).
* Explored electric range performance across vehicle models.
* Evaluated Clean Alternative Fuel Vehicle (CAFV) eligibility patterns.

## 🛠️ Tech Stack

* Tableau
* SQL
* Python
* Pandas
* Data Visualization

## 📂 Dataset Attributes

* Vehicle Make
* Vehicle Model
* Model Year
* Electric Range
* Vehicle Type
* State
* County
* CAFV Eligibility
* Electric Utility
